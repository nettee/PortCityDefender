# 港城卫士

本项目是一款支持iOS/Android平台的信息交流应用。

### 开发所用技术

+ 使用**[Ionic](http://ionicframework.com/)**混合移动应用开发框架进行前端开发
+ 使用Node.js模块**[Express](http://expressjs.com/)**和文档型数据库
**[MongoDB](https://www.mongodb.org/)**搭建服务器
