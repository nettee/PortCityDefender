# 1引言

## 1.1编写目的

本文档是软件测试分析报告，编写本文档意在：

* 通过分析软件的测试过程，判断测试流程是否与软件测试计划相一致。

* 通过分析软件的测试结果，评估软件的质量。

本文档的预期读者包括：软件质量保障人员、软件测试工程师、软件发布与维护人员。

# 3测试内容及结果

## 3.1客户端测试

a.登陆

| 输入/操作 | 预期输出/响应 | 实际输出/响应 | 备注 |
|---|---|---|---|
| 点击手机应用图标 | 应启动应用进入登陆界面，登陆界面以“港城卫士”为标题，从上到下包含用户名输入框、密码输入框和登陆按钮 | 启动应用进入登陆界面，登陆界面与预期相同  |  | 
| 输入密码 | 密码应显示为“*” | 密码显示为“*” |
| 输入正确的用户名和密码，点击登陆按钮 | 应登陆成功进入情报页面 | 登陆成功进入情报页面 |
| 输入错误的用户名或密码，点击登陆按钮 | 应弹出提示框“用户名或密码错误”，点击确定后返回登陆界面 | 弹出提示框“用户名或密码错误”，点击确定后返回登陆界面 | 
| 不输入用户名或密码 | 登陆按钮应为浅色，不可点击 | 登陆按钮为浅色，不可点击 |

| 点击“情报动态” | 应进入情报动态页面，显示情报动态列表，同时左侧菜单消失 | 进入情报动态页面，显示情报动态列表，同时左侧菜单消失 |
| 点击“控制指挥” | 应进入控制指挥页面，显示收到的命令列表，同时左侧菜单消失 | 进入控制指挥页面，显示收到的命令列表，同时左侧菜单消失 |

c.个人信息

| 输入/操作 | 预期输出/响应 | 实际输出/响应 | 备注 |
|---|---|---|---|
| 在左侧菜单栏中点击个人头像或介绍区域 | 应进入个人信息界面，界面从上到下应显示个人头像、个人介绍、用户名、区域、电话、等级信息和修改密码、退出登录按钮 | 进入个人信息界面，界面现实与预期相同 |
| 在个人信息页面，点击修改密码 | 应进入修改密码页面，显示旧的密码、新的密码、确认密码三个输入框和确认修改按钮 | 进入修改密码界面，界面显示与预期相同 |
| 在修改密码界面，点击左上角的返回箭头 | 应返回个人信息页面 | 返回个人信息页面 |
| 在修改密码界面，输入错误的原密码，点击确认修改按钮 | 应出现“原密码错误”的提示框，点击“确定”，返回当前页面 | 出现“原密码错误”的提示框，点击“确定”，返回当前页面 |
| 在修改密码界面，输入正确的原密码，新密码和确认密码输入不同的值，点击确认修改按钮 | 应出现“确认密码错误”的提示框，点击“确定”，返回当前页面 | 出现“确认密码错误”的提示框，点击“确定”，返回当前页面 |
| 在修改密码界面，输入正确的原密码，新密码为空，点击确认修改按钮 | 应出现“修改密码成功”的提示框，点击“确定”，返回到登录界面 | 出现“修改密码成功”的提示框，点击“确定”，返回到登录界面 |
| 在个人信息页面，点击退出登录按钮 | 应出现确认框，点击“确定”，退出登录并返回登录页面；点击“取消”，返回当前界面 | 出现确认框，点击“确定”，退出登录并返回登录页面；点击“取消”，返回当前界面 |

d.控制指挥

| 输入/操作 | 预期输出/响应 | 实际输出/响应 | 备注 |
|---|---|---|---|
| 左侧菜单中点击指挥控制 | 应进入指挥控制页面，出现用户收到的命令列表和用户发送的命令列表，连个列表均可为空，当列表为空时，列表中内容用“暂无命令”替代 | 进入指挥控制页面，页面内容与预期相同 |
| 下拉刷新“指挥控制”页面 | 应显示最新的命令列表 | 显示最新的命令列表 |
| 指挥控制页面中，在左侧菜单先点击其他项，再次点击指挥控制 | 应进入指挥控制页面，显示最新的命令列表 | 进入指挥控制页面，显示最新的命令列表 |
| 指挥控制页面中，点击任意一条命令 | 应进入命令详情页面，详情中的头像，发送者，时间应该与点击前命令列表中的该命令一致。 | 进入命令详情页面，详情页面显示与预期相同 |
| 指挥控制页面中，点击右上角的“+” | 应进入新建命令页面，且新建命令页面接收者处为空，内容输入处默认显示“请输入...” | 进入新建命令页面，新建命令页面接收者处为空，内容输入处显示“请输入...” |
| 命令详情页面中，点击左上角的指挥控制 | 应返回指挥控制页面，且显示最新的命令列表 | 返回指挥控制页面，显示最新的命令列表 |
| 新建命令页面中，内容输入处输入内容，观察每行字数和行数 | 每行字数到达输入框的边界时应该换行，字的行数超过输入框的高度时可用手滑动屏幕调整 | 字数到达输入框的边界时自动换行，可用手滑动屏幕调整 |
| 新建命令页面中，按左上角的指挥控制 | 应回到命令列表的页面，且显示最新命令列表 | 回到命令列表的页面，显示最新命令列表 |
| 新建命令页面中，接收者为空，按发送按钮 | 应弹出提示“接收者为空！”，按确定后仍停留在新建命令页面 | 弹出提示“接收者为空！”，按确定后停留在新建命令页面 |
| 新建命令页面中，接收者不为空，内容输入处为空，按发送按钮 | 应弹出提示“命令内容为空”，按确定后仍停留在新建命令页面 | 弹出提示“命令内容为空”，按确定后停留在新建命令页面 |
| 新建命令页面中，接收者，内容输入处均不为空，按发送按钮 | 页面应跳转到指挥控制页面，且显示最新的命令列表 | 页面跳转到指挥控制页面，同时显示最新的命令列表 |
| 新建命令页面中，点击联系人图标 | 应进入选择联系人页面，默认联系人列表按照区域展开，联系人应为当前用户的直接下级，搜索框为空。 | 进入选择联系人页面，页面显示与预期相同 |
| 选择联系人页面中，点击区域 | 点击前后的区域列表折叠与否应该恰好相反 | 区域列表可以折叠 |
| 选择联系人页面中，点击联系人前面的勾选框 | 联系人的选中状态应发生变化，选中后选择框中有对勾，未选中对勾消失 | 联系人的选中状态发生变化，选中后选择框中有对勾，未选中对勾消失 |
| 选择联系人页面中，点击确定 | 页面跳转到新建命令页面，新建命令页面中接收者应该与选择联系人页面中的选择保持一致 | 页面跳转到新建命令页面，新建命令页面中接收者与选择的联系人保持一致， |
| 选择联系人页面中，点击取消 | 页面应跳转到新建命令页面 | 页面跳转到新建命令页面 |
| 选择联系人页面中，在搜索框中输入姓名或者id | 在搜索框下方应出现姓名或id中含输入内容的联系人 | 出现姓名或id中含输入内容的联系人 |
